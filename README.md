# Neon Oort - AI Quiz Platform

This requires a local development server to run effectively because uses modern JavaScript ES modules.

## How to Run (Recommended)

1.  Open this folder in VS Code or Terminal.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Click the link shown in the terminal (usually `http://localhost:5173`).

## Troubleshooting

-   **White/Blue Screen**: If you just double-click `index.html`, you will see a blank screen or just the background. **You MUST run `npm run dev`**.
-   **Loading...**: If it stays on "Loading...", check your valid javascript enabled browser console (F12) for errors.
